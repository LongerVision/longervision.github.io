Calibration based on Various Patterns/Markers from OpenCV

First of all, we need to ensure cv2.so is included under our system folder. This cv2.so is specifically for OpenCV Python.

Then, some standard packages need to be imported. (Due to your own requirements)

Afterwards, we load a dictionary to be used. Current OpenCV provides four groups of aruco patterns, 4X4, 5X5, 6X6, 7X7, etc. Here, aruco.DICT_6X6_1000 is arbitrarily selected as an example.
