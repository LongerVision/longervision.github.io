Posted on 2023-12-04 Edited on 2026-01-21 In Single Board Computers , Risc-V

Long time no see everybody. Today let’s talk about MaixDuino board.

1. Introduction

1.2 ll /dev/ttyUSB*

1.3 Hardware Wiki

Detailed specification about MaixDuino board can be found on MaixDuino Wiki.

The Maixduino is equipped with two onboard chips:

  • a Sipeed M1 AI module featuring the Kendryte K210, and
  • an ESP32-WROOM-32.

Specifically, Sipeed M1 comes with:

  • RISC-V RV64GC 400~600MHz
  • CPU DUAL CORE 64BIT
  • KPU 230GMULps 16BIT
  • FPU DP8SP
  • APU 8MIC

1.4 Github Source Code

Typically, there’s no need to rebuild your project from the ground up. Instead, simply enter the Maixduino package URL in the Additional Boards Manager URLs section.

However, when needed, Maixduino Github repository offers open-source resource that you can utilize as a foundation for further development.

1.5 Three Meanings Of Maixduino

  • MaixDuino board: the development board itself
  • Maixduino Arduino Board Manager support: Add https://dl.sipeed.com/MAIX/Maixduino/package_Maixduino_k210_index.json as one Additional Board Manager URL
  • Maixduino Official Documentation

Note: Maixduino supports Maix Bit board.
